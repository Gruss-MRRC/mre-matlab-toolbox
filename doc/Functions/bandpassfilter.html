bandpassfilter

function f = bandpassfilter(sze, cutin, cutoff, n)

Creates bandpass Butterworth filter in two dimensions. BANDPASSFILTER - Constructs a band-pass butterworth filter usage: f = bandpassfilter(sze, cutin, cutoff, n) where: sze is a two element vector specifying the size of filter to construct. cutin and cutoff are the frequencies defining the band pass 0 - 0.5 n is the order of the filter, the higher n is the sharper the transition is. (n must be an integer >= 1). The frequency origin of the returned filter is at the corners. See also: LOWPASSFILTER, HIGHPASSFILTER, HIGHBOOSTFILTER
