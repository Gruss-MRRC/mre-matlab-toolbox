function im_unwrapped = QualityGuidedUnwrap2D_r1(im_mag,im_phase)

Implements 2D quality guided path following phase unwrapping algorithm. Inputs: 1. Complex image in .mat double format 2. Binary mask (optional) Outputs: 1. Unwrapped phase image 2. Phase quality map This code can easily be extended for 3D phase unwrapping. Technique adapted from: D. C. Ghiglia and M. D. Pritt, Two-Dimensional Phase Unwrapping: Theory, Algorithms and Software.
